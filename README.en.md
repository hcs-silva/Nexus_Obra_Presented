# Nexus Obra - Startup Technical Portfolio

Language: [PT](README.md) | **EN**

This directory presents Nexus Obra as a **construction-tech startup** balancing **product value** and **engineering rigor**.

## What is included

- Visual public landing page: `index.en.html`
- Styles and interactions: `styles.css`, `script.js`
- Technical supporting documentation: `docs/`
- Visual assets reused from the product: `assets/img/`
- Portuguese version: `index.html` and `README.md`

## Portfolio narrative

- Real problem: construction teams need operational and financial control in one workflow.
- Solution: Nexus Obra integrates planning, budgeting, field operations, and monitoring.
- Technical evidence: architecture, security, delivery quality, and BOQ contingency case study.
- Startup focus: execution speed, decision clarity, and multi-tenant scalability.

## Technical capability demonstration

- Full-stack architecture with end-to-end TypeScript for domain consistency.
- Modular backend with authentication, authorization, observability, and rate-limit middleware.
- Data management with revision-oriented models for decision traceability.
- Delivery quality with unit tests, E2E tests, and disciplined release practices.
- Product operations with smoke-check scripts, readiness reports, and CI workflows.
